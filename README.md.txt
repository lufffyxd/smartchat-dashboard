# SmartChat Dashboard

A modern AI-powered dashboard application with authentication, chat windows, notes, and task management.

## Features

- User authentication (Sign Up / Sign In)
- Dashboard with multiple AI windows
- Chat interface with web search and news integration
- Notes editor with text selection features
- Task manager
- Automatic news fetching
- Pin up to 3 windows
- Responsive design
- Dark theme with improved text visibility
